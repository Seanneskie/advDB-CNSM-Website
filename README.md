# advDB-CNSM-Website


## Introduction - How to Install

### Setup backend node modules
```bash
cd backend
npm install
```

### Setup frontend modules
```bash
cd frontend
npm install
```

## How to Run
### 1. Run Backend Server
Without Nodemon Method
```bash
cd backend
npm run dev
```
Nodemon Method
```bash
cd backend
nodemon run dev
```

### 2. Run Frontend Server
```bash
cd frontend
npm start
```

### Open Chrome and go to http://localhost:3000/
